# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Pianel, please report it privately rather
than opening a public issue.

- Use GitHub's **[Private vulnerability reporting](https://github.com/pianelapp/pianel/security/advisories/new)**
  (Security tab → "Report a vulnerability"), or
- Contact the maintainers at **security@pianel.app**.

Please include steps to reproduce, affected platform (mobile / desktop / web),
and any relevant logs. We will acknowledge your report as soon as possible and
keep you informed as we work on a fix.

## Scope

Pianel is an offline-first application whose only external connection is
Bluetooth to the piano. Reports involving the BLE/MIDI handling, preset
import/export parsing, and dependency vulnerabilities are especially welcome.

## Supported Versions

As an early-stage project, security fixes are applied to the latest `main`.
Once tagged releases exist, this section will list supported versions.
